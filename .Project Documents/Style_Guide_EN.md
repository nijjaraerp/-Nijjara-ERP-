# Style Guide (English)

## Typography
- Cairo family: 400 Regular, 600 SemiBold, 700 Bold
- Header: 48px title (Cairo 700), 18px subtitle (Cairo 400)

## Inputs & Button
- Inputs: 48px height, 16px padding, 2px radius
- States: normal, focus, error with clear visual cues
- Button: "تسجيل الدخول" label, Cairo 600 18px, min-width 200px, height 48px

## RTL
- Arabic layout uses RTL; fields align right

## Colors
- Primary: #1e3a8a; Hover: #2b4db5; Focus: #3b82f6; Error: #dc2626

## Accessibility
- WCAG 2.1 AA contrast
- Visible focus ring
- Touch targets >=48x48px
- Prefers-reduced-motion support
